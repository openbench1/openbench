# Privacy Policy

**Last Updated: February 2026**

## Introduction

OpenBench ("we", "our", "us") operates the website [openbench.xyz](https://www.openbench.xyz). This Privacy Policy explains how we collect, use, and protect your information.

## Information We Collect

### Automatically Collected
- **Contract Addresses**: Addresses you submit for scanning (publicly available on-chain data)
- **Usage Data**: Page views, feature usage, and scan history (via anonymous analytics)

### Wallet Connection
- When you connect a Web3 wallet, we access your **public wallet address** only
- We **never** request or store private keys, seed phrases, or signing permissions beyond transaction approval

## Information We Do NOT Collect
- Personal identification (name, email, phone)
- IP address tracking or fingerprinting
- Private keys or seed phrases
- Browsing history outside our platform

## How We Use Information
- Provide security scanning and audit services
- Improve detection accuracy and platform performance
- Display scan history in your dashboard (stored locally or linked to wallet address)

## Data Storage
- Scan results are stored in our database for service functionality
- No personal data is sold or shared with third parties
- All API communications are encrypted via HTTPS

## Third-Party Services
- **GoPlus Security API**: For token risk analysis (receives contract addresses only)
- **OpenAI / Anthropic API**: For AI code audit (receives smart contract source code only)
- **Etherscan / BSCScan API**: For fetching verified contract source code

## Your Rights
- You may disconnect your wallet at any time
- Scan data associated with your wallet can be deleted upon request
- Contact us via [Twitter/X](https://x.com/openbench_lab) for data requests

## Changes
We may update this policy periodically. Changes will be posted on this page with an updated date.
